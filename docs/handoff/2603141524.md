### Commit 2 (2603141516) — 대시보드 UI 대폭 개선 — 타임라인, 문서 모달, 헤더 재설계

실시간 피드백을 바탕으로 대규모 UI 반복 작업.

- **스트림라인**: 카드 중앙을 관통하는 세로선 (z-index를 통해 뒤로 통과); JS가 첫 번째/마지막 카드 중앙점에서 정확한 시작/끝 계산; 카드 펼침/접힘 시 업데이트
- **현재 세션 카드**: `git log --after=<마지막 핸드오프 타임스탬프>`를 읽어 핸드오프되지 않은 커밋 표시; 강조 테두리
- **핸드오프 카드**: 접힘 = 날짜 + 모든 커밋 요약 (각 한 줄, 잘림); 펼침 = 전체 커밋 상세 + 다음 작업
- **문서 모달**: 프로젝트별 `[docs]` 아이콘 버튼; ROADMAP / README / ARCHITECTURE 탭; Python `markdown` 라이브러리를 통한 서버 사이드 마크다운 렌더링; `/api/doc/<project>/<doctype>` 엔드포인트
- **컬럼 헤더 (고정)**: 프로젝트명 + docs 아이콘 + Phase N 정보 + 프로그레스 바 (6px) + 목표 (40px 고정, line-clamp:2) + 작업 (40px 고정, 행당 flex-shrink:0); 헤더 클릭으로 목표+작업 토글; 30초 리렌더 간 상태 유지
- **빈 핸드오프 건너뜀**: 커밋 0개인 이전 한국어 형식 핸드오프 렌더링하지 않음
- **너비 수정**: `.col`에 `max-width + overflow:hidden`으로 가변 너비 방지
