### Commit 1 (2603151211) — Post-it notes feature for idea capture

Added a sticky-note system to the dashboard for capturing project ideas and free-form memos.

**Backend:**
- `data/ideas.json` — persistent storage for notes (position, color, project ownership)
- Ideas CRUD API: `GET/POST /api/ideas`, `PUT/DELETE /api/ideas/<id>`
- Promote API: `POST /api/ideas/<id>/promote` — scaffolds a new project folder with ROADMAP.md + README.md from note content (currently disabled in UI)

**Frontend:**
- Draggable post-it notes overlaid on the board (`position: absolute` in a `#notes-layer`)
- 4 color themes: yellow, pink, blue, green
- Double-click to expand/collapse; expanded state shows title input, description textarea, color dots, delete button
- Auto-save on blur (title and description)
- Project ownership detection: on drag-end, checks which column the note center overlaps and assigns `project` field; column highlights with dashed outline during drag
- `+ Note` button in header to create new notes
- `ESC` key to collapse expanded note (document-level `keyup` for IME compatibility)
- `Ctrl+M` shortcut to create new note

**Structure change:** `#board` wrapped in `#board-wrapper` (position: relative) to serve as positioning context for absolute-positioned notes.
