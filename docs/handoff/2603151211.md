## 이번 세션

### 커밋 1 (2603151211) — 아이디어 캡처를 위한 포스트잇 노트 기능

대시보드에 프로젝트 아이디어와 자유 형식 메모를 기록하기 위한 스티키 노트 시스템 추가.

**백엔드:**
- `data/ideas.json` — 노트 영구 저장소 (위치, 색상, 프로젝트 소속)
- 아이디어 CRUD API: `GET/POST /api/ideas`, `PUT/DELETE /api/ideas/<id>`
- 프로모트 API: `POST /api/ideas/<id>/promote` — 노트 내용으로 새 프로젝트 폴더에 ROADMAP.md + README.md 스캐폴딩 (현재 UI에서 비활성화)

**프론트엔드:**
- 보드 위에 오버레이되는 드래그 가능한 포스트잇 노트 (`#notes-layer` 내 `position: absolute`)
- 4가지 색상 테마: 노랑, 분홍, 파랑, 초록
- 더블클릭으로 펼침/접힘; 펼친 상태에서 제목 입력, 설명 텍스트영역, 색상 점, 삭제 버튼 표시
- 블러 시 자동 저장 (제목 및 설명)
- 프로젝트 소속 감지: 드래그 종료 시 노트 중심이 겹치는 컬럼을 확인하여 `project` 필드 할당; 드래그 중 컬럼에 점선 외곽선 강조
- 헤더에 `+ Note` 버튼으로 새 노트 생성
- `ESC` 키로 펼쳐진 노트 접기 (IME 호환을 위한 문서 레벨 `keyup`)
- `Ctrl+M` 단축키로 새 노트 생성

**구조 변경:** `#board`를 `#board-wrapper` (position: relative)로 감싸 absolute 위치 노트의 포지셔닝 컨텍스트 역할 수행.
